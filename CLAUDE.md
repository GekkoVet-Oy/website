# GekkoVet Marketing Website

Marketing website for www.gekkovet.com.

## Tech Stack

- **Static site generator:** Hugo (extended)
- **Hosting:** GitHub Pages
- **Repo:** GekkoVet-Oy/website

## Commands

- `hugo server` — local dev server with live reload
- `hugo` — build to `public/`
- `hugo --minify` — production build (used in CI)

## Deployment

Automatic via GitHub Actions on push to `main`. See `.github/workflows/deploy.yml`.
